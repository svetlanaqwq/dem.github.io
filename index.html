Из чего состоит доменное имя?

Дата публикации: 1 января 1996 г.

Рассмотрим доменное имя моего сайта:

https://svetlanaqwq.github.io/dem.github.io/

На самом деле, основной хост (домен) здесь — это:

svetlanaqwq.github.io

Разберём его по уровням иерархии DNS (справа налево):

  • .io — домен верхнего уровня (TLD, Top-Level Domain). Изначально принадлежал Британской территории в Индийском океане, но сейчас широко используется в IT-сфере.
  • github — домен второго уровня (SLD, Second-Level Domain). Принадлежит компании GitHub и используется для бесплатного хостинга сайтов через GitHub Pages.
  • svetlanaqwq — поддомен третьего уровня. Это имя вашего аккаунта на GitHub. Именно оно делает ваш сайт уникальным в рамках github.io.

Часть /dem.github.io/ — это не домен, а путь к ресурсу (каталог или файл) на сервере. Она не влияет на DNS-иерархию.

Таким образом, полное доменное имя (FQDN) вашего сайта: svetlanaqwq.github.io.
